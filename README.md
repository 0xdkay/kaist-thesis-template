# KAIST Thesis Template

## Usage

1. clone the git repo:
  ```
  $ git clone git@github.com:0xdkay/kaist-thesis-template.git
  ```

2. build it:
  ``` sh
  $ make

  (check p.pdf)
  ```

3. starting point, p.tex:
  ```
  $ grep -oP 'input{\K\w+' p.tex
  pkgs
  macro
  abstract
  intro
  back
  anal
  accounting
  impl
  eval
  attack
  cm
  related
  concl
  summary
  ack
  cv
  ```
